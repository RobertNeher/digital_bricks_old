# digital_bricks

Simulation of logical circuits.

## Features
** Pre-defined logic circuits: AND, OR, XOR, NAND, NOR, NXOR: Each with flexible number of inputs
** Flip-flop
** Oscillator
** 7 or 16 segment display
** keyboard entry (4, 7 or 8 bit)
** D/A converter (4, or 8 bit)
** loud speaker
** LED (1 to n with Color range(s))
** Button (double or triple state)
** touch pad (mouse replacement) with 4, or 8 bit resolution
